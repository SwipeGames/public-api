| EUR           | European member countries Euro           |
| IDR           | Indonesia Rupiah                         |
| INR           | India Rupee                              |
| JPY           | Japan Yen                                |
| KRW           | Korea (South) Won                        |
| MXN           | Mexico Peso                              |
| RUB           | Russia Ruble                             |
| THB           | Thailand Baht                            |
| TRY           | Turkey Lira                              |
| TWD           | Taiwan New Dollar                        |
| USD           | United States Dollar                     |
| UZS           | Uzbekistan Som                           |
| VND           | Vietnam Dong                             |
